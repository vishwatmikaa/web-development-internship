Week 1 Assignment: Portfolio Webpage 

Project Overview
This is my Week 1 project submission for the BeeSkilled internship. The goal of this assignment was to build a clean, structural, and modern single-page personal portfolio from scratch using semantic HTML and custom styling with CSS. 

 Project Preview



Verification Criteria Met

As requested by the assignment guidelines, this project successfully implements and demonstrates the following core skills:

 1. Four Sections
The webpage is cleanly organized into the four required sections using internal anchor links for smooth navigation:
 About: A professional summary introducing myself as a CSE student.
 Education: Structural layout cards displaying my academic track at Institution and Percentage.
 Skills: Grouped categorical breakdowns highlighting Frontend, Software Engineering, and System Foundations.
 Contact: A dedicated section allowing evaluators and recruiters to connect via email.

 2. Semantic HTML5 Tags
Instead of using generic block elements everywhere, the code is structured using modern, accessible HTML5 semantic tags:
* `<header>` and `<nav>` to wrap the sticky top navigation menu.
* `<main>` to contain the core content of the application.
* `<section>` to isolate each individual part of the portfolio cleanly.
* `<footer>` to hold the copyright text.

3. Profile Photo & Basic Styling
Profile Photo: Embedded `profile.jpg` in the About section, custom-styled into a clean circular profile frame with an overlay shadow effect.
Layout Design: Used CSS Flexbox properties to handle alignment between text elements and images.
UI Details: Added custom border highlights (`border-left`) to education cards, and converted boring bullet lists into beautiful rounded skill tags.
